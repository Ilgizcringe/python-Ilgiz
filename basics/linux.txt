Линукс команды

1)  pwd (print working directory) ->  возвращает абсолютный путь до папки (показывает где мы находимся)

2)  ls (list) -> выводит содержимое директории (список файлов и папок текущей директории)
	2.1 ls -a -> выводит все файлы и директории, включая скрытые
	2.2 ls -l -> выводит все содержимое (кроме скрытых) с дополненительной инф
	2.3 ls -la -> выводит все содержимое, включая скрытые директории и файлы с дополнительной информацией
3)cd / cd ~ (change directory) -> переход в домашнюю директорию
	3.1 cd <dirname> -> переход в директорию dir_name
	3.2 cd /home/user/dir/dir -> переход в директорию по абсолютному пути
	3.3 cd .. -> подняться на уровень выше (вернуться к предыдущей директории)
4)mkdir (make directory) -> создаёт директорию
	mkdir <name>
5)touch <filename> ->  создаёт указанный файл
6)rmdir (remove directory) -> удаляют пустую директорию 
	rmdir <dir_name>
7)rm -rf (remove recursive force) -> принудительное удаление, удаляет директории с содержимым
8)rm (remove) -> удаляет файл
	rm <filename>
9)clear -> очищает терминал
10)cp (copy) -> копирует файл или директорию в другую директорию
	cpv<что копируем filename> <куда копируем dir_name>
11)mv (move) -> 1.перемещает файл
		mv <filename> <dir_name>
	mv -> переименовывает 
		mv <filename> <new_filename>
12) nano -> создаёт и открывает файл в встроенном текстовом редакторе
13) echo -> записывает содержимое в файл
	echo текст > filename
14)cat-> выводит содержимое файла в терминал
	cat <filename>
  cat -> записывает содержимое файла в другой файл
	cat filename > filename2
15)history -> выводит историю команд
16)du (disk user) -> выводит информацию о занимаемой памяти
17) find -> ищет файл или директорию
	file /home -name <filename/dtr_name> -> выводит абсолютный путь
18)sudo -> предоставляет права администратора (выполняет команду с правами администратора)
19) code . -> открывает текущую директорию в vscode


ctrl+alt+T -> открывает терминал
Нажатие Tab -> приводит к автозаполению 
Стрелками вверх.вниз -> можно перемещать по использованным командам
Чтобы закрыть текстовый редактор nano -> ctrl + S >> ctrl + X 
					ctrl + X >> Y >> Enter
